3. Generates v2ray config with SOCKS5 (10808) + HTTP (10809) inbounds
4. Adds routing rules: CN traffic → direct, international → proxy

4. Installed via: `sudo dpkg -i installer_debian_x64_2.2.7.5.deb`
5. Created symlink: `sudo ln -sf ~/.local/bin/v2ray /usr/local/bin/v2ray` (so v2rayA can find the v2ray binary)

- Web UI: `http://localhost:2017`
- The old user-level `v2ray.service` was stopped and disabled to avoid conflict
